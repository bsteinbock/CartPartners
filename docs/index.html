CartPartners

Version 1.0

CartPartners is a mobile application designed to help you stay connected with your golfing buddies by ensuring that your cart partners change from round to round. The app supports organizing players into leagues or outings, managing player availability, creating rounds, generating tee-time groups, and sending notifications to players via email or text message.

App Features

CartPartners provides a comprehensive set of features organized into tabs and drawer screens for easy navigation and management of golf outings.

Rounds Tab

Displays all your defined rounds and includes a button to add a new round.

  • League/Outing Picker — Select a specific league or outing at the top of the screen
  • Long-press a round to edit its details (course, date, and tee-time information)
  • Swipe left on a round to delete it and all associated data
  • Tap a round to open its Lineup tab

Lineup Tab

Specify the players participating in a particular round.

  • Round Selection — Use the picker at the top to switch between rounds
  • View all golfers whose status is set to "available"
  • Tap the icon at the top right to add players from the master player list
  • Toggle player participation for the selected round
  • Press the Groups icon at the bottom to open the Groups tab when the lineup is finalized

Groups Tab

Create and manage the tee-time groupings for a round.

  • Generate — Create groups for the first time
  • Regenerate — Update existing groups; the algorithm ensures cart partners change from round to round
  • Select a group to access edit and reordering options
  • Move groups up or down in the tee-time order
  • Use the icon at the top right to manually adjust players in any group
  • Swap players between groups as needed
  • Use the airplane icon to email all players their group assignments

More Tab — Drawer Screens

The More tab provides access to additional screens through a drawer navigation panel. Open the drawer by tapping the icon at the top left of the screen.

Notify Players

Send notifications to players via email or text message.

  • Player Selection — Choose which players to notify from the current league's player list
  • Message Composition — Enter a custom title and message
  • Delivery Options — Send via email or SMS (when available)

Leagues / Outings

Manage multiple leagues or outings, each with its own set of players and rounds.

  • Create — Add new leagues or outings with custom names
  • Long-press to edit league details
  • Swipe left to remove a league
  • Maintain separate player lists and rounds for different groups

League Players

Define the list of players specific to a particular league or outing.

  • Add players from the master player list to the current league
  • Swipe to remove players from the league (does not delete from the master list)
  • Export the league's player list to a CSV file
  • Choose which league to manage using the picker at the top

Manage All Players

Manage the master player list containing all players known to the app.

  • Add new player entries with name, email, and phone information
  • Edit player details including availability status
  • Delete players from all leagues, rounds, and groups
  • Import players from a CSV file
  • Export the entire player list to CSV for backup or sharing

Backup / Restore

Create and restore database backups containing all CartPartners data.

  • Backup — Create a database file with all your leagues, players, rounds, and groups
  • Restore — Load data from a previous backup file
  • Sharing — Share backup files with another person taking over as Group Coordinator
  • Warning: Restoring a backup will overwrite all existing data in the app
